SlTexture * SlManager::createTextureFromTile ( std::string  name,
std::string  sprite,
int  width,
int  height 
)

Creates a new texture with dimensions w x h and fills it with tiles of SlSprite tile.
Creates SlSprite of same name that holds the whole texture.

The SlRenderSettings::destinationRect at position 0 in the SlSprite::destinations_ is used to determine the width of the tile. If the SlSprite::destinations_ is empty, addDefaultDestination() is called which sets the destinationRect to equal the sourceRect.
